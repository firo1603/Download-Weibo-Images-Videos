# Download Weibo Images & Videos（个人维护版）

基于原项目：
- https://github.com/owendswang/Download-Weibo-Images-Videos

当前仓库在保持原有功能的基础上，做了适配和增强，方便在新版微博页面继续稳定使用。

## 当前版本

- Userscript 版本：`1.5.1.1`

## 主要特性

- 支持微博新版 UI 的图片、视频下载
- 支持单条动态下载、单图下载
- 支持 ZIP 打包下载
- 支持 Aria2 RPC 远程下载
- 支持下载视频封面、去水印图片（可选）
- 支持保存微博文字内容为 `.txt`（可选）

## 本仓库增强项

- 显示 ZIP 打包进度
- 下载进行中时，切换标签页尽量保持后台下载活跃
- 下载按钮样式优化（含状态展示）
- 相册页能力增强：
	- 单图下载按钮
	- 下载全部（支持点击停止）


## 安装方式

1. 浏览器安装 Tampermonkey（或同类用户脚本扩展）
2. 新建脚本，将 [Download Weibo Images & videos.user.js](Download%20Weibo%20Images%20%26%20videos.user.js) 全部内容粘贴保存
3. 打开微博页面刷新即可生效

## 使用说明

- 页面左侧会出现设置按钮，可配置命名规则、ZIP、Aria2、是否保存文字等
- 动态卡片会出现下载按钮；多图动态支持单图下载
- 个人主页相册页支持“下载全部 / 停止下载”

## 说明

- 仅用于个人学习与备份，请遵守微博平台规则及相关法律法规
- 如果微博页面结构变化，脚本可能需要跟进调整
